Array Methods Demonstration

Map

Transforms each element in an array and returns a new array.

const doubled = numbers.map(num => num * 2);

Filter

Creates a new array with elements that pass the test.

const evenNumbers = numbers.filter(num => num % 2 === 0);

Reduce

Reduces an array to a single value by applying a function.

const sum = numbers.reduce((total, num) => total + num, 0);

ForEach

Executes a function once for each array element.

let output = [];
numbers.forEach(num => output.push(num * 3));

Splice

Changes the contents of an array by removing or adding elements. 

const spliced = [...numbers]; // Create a copy
spliced.splice(5, 1, 99);
